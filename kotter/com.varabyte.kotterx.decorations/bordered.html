fun RenderScope.bordered(borderCharacters: BorderCharacters = BorderCharacters.BOX_THIN, paddingLeftRight: Int = 0, paddingTopBottom: Int = 0, render: OffscreenRenderScope.() -> Unit)

Automatically render a border around some inner content.

Parameters

borderCharacters

The characters used to render the border.

paddingLeftRight

If set, adds some additional padding at the start and end of every line.

paddingTopBottom

If set, adds some newlines before and after the entire block of text.

render

The render block that generates content (e.g. via textLine) which will be wrapped within a border.
