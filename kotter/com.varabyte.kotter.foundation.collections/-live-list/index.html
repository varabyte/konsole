LiveList

@ThreadSafe
class LiveList<T> : MutableList<T>

Like LiveVar, but for lists.

In other words, adding or removing to the list will cause the active section to rerender automatically.

For example:

val ids = liveListOf<String>()

section {
  for (id in ids) { ... }
}.run {
  ids.add(...)
}
