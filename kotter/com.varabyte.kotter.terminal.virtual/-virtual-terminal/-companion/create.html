fun create(title: String = "Virtual Terminal", terminalSize: TerminalSize = TerminalSize.Default, fontSize: Int = 16, fontOverride: Path? = null, fgColor: Color = AnsiColor.WHITE, bgColor: Color = AnsiColor.BLACK, linkColor: Color = AnsiColor.CYAN, maxNumLines: Int = 1000, handleInterrupt: Boolean = true): VirtualTerminal

Factory method for constructing a VirtualTerminal.

Parameters

title

The text to use for the terminal window's title bar.

terminalSize

Number of characters, so 80x32 will be expanded to fit 80 characters horizontally and 32 lines vertically (before scrolling is needed)

fontSize

The size to use for the font used by this virtual terminal.

fontOverride

A path to a font file resource (e.g. ttf) for an alternate text look and feel.

fgColor

The color to use for the font text.

bgColor

The color to use for the virtual terminal background color.

linkColor

The color to use for links.

maxNumLines

The number of text lines to keep before truncating the oldest ones. This can be useful to ensure that this program won't eventually run out of memory if you keep appending text in a loop forever. This value Will be clamped to at least TerminalSize.height. Set to Int.MAX_VALUE if you don't want truncation to happen.

handleInterrupt

If true, handle CTRL-C by closing the window.
