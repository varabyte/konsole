SectionScope

interface SectionScope

Common interface used for scopes that can appear in both the render and run blocks.

For example:

section {
  ... // A
}.run {
  ... // B
}

// A is a RenderScope and ALSO a SectionScope
// B is a RunScope and ALSO a SectionScope
