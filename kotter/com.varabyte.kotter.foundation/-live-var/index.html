LiveVar

@ThreadSafe
class LiveVar<T>

A special variable which can be used to auto-rerender a target Section without needing to call RunScope.rerender yourself.

For example:

var count by liveVarOf(0)
section {
   for (i in 0 until count) { // <-- LiveVar read happens here
     text("*")
   }
}.runUntilFinished {
  while (count < 5) {
    delay(1000)
    ++count // <-- LiveVar write happens here; rerender is triggered
  }
}

count = 123 // Setting LiveVar out of a section is harmless; no rerender is triggered
